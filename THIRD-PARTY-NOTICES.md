# 함께 배포되는 폰트

[`LICENSE`](./LICENSE) 의 MIT 라이선스는 yulyrics **소스 코드**에 적용됩니다.
프로그램에 포함된 폰트는 MIT 가 아니라 **각 폰트의 라이선스**를 따릅니다.

아래 폰트는 모두 **SIL Open Font License 1.1** 이며 재배포가 허용됩니다.
라이선스 원문은 [`packages/presets/fonts/licenses/`](./packages/presets/fonts/licenses/) 에 폰트별로 들어 있습니다.

| 폰트 | 저작권 | 출처 |
|---|---|---|
| Jua (주아) | The Jua Project Authors | https://fonts.google.com/specimen/Jua |
| Do Hyeon (도현) | The Do Hyeon Project Authors | https://fonts.google.com/specimen/Do+Hyeon |
| Black Han Sans (검은고딕) | The Black Han Sans Project Authors | https://fonts.google.com/specimen/Black+Han+Sans |
| Bagel Fat One | The Bagel Fat Project Authors | https://fonts.google.com/specimen/Bagel+Fat+One |
| Pretendard | Kil Hyung-jin | https://github.com/orioncactus/pretendard |
| Montserrat | The Montserrat Project Authors | https://fonts.google.com/specimen/Montserrat |

## 폰트를 추가하실 때

로컬 서버가 폰트 파일을 OBS 로 내려보내는 구조라, 폰트를 넣는 것은 **명백한 재배포**입니다.
"무료 사용 가능"과 "재배포 가능"은 다릅니다 — 재배포가 허용되지 않는 폰트는 넣지 마세요.

PC에 설치된 폰트를 쓰고 싶다면 프리셋의 폰트 이름에 시스템 폰트명을 직접 적으면 됩니다.
그건 재배포가 아니므로 라이선스 문제가 없습니다.
