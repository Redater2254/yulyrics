## preset-1-karaoke.png 시각 분석 (원본 2538×745 추정, 가로로 긴 크롭)

노래방/가라오케 스타일. 배경은 알록달록한 보케(bokeh) 영상 — **배경은 참고용이며 오버레이 대상이 아님**.

**레이아웃**
- 2줄 교차 배치: 1행은 화면 **좌측 정렬(상단)**, 2행은 화면 **우측 정렬(하단)**
- 각 가사 줄 바로 아래에 **로마자 발음(romaja)** 보조 줄이 붙음 → 실제로는 4줄 스택
- 메인 줄과 보조 줄의 크기 비율 대략 **1 : 0.55**

**메인 가사 (한글)**
- 둥근 고딕 계열 볼드 (배민 주아 / 카페24 써라운드 / 야놀자 야체 류의 라운드 폰트)
- 채움색: 연하늘 `#B8DFFF` 근처
- 외곽선: 진한 로열블루 `#1B4FD8` 근처, 매우 두꺼움 (폰트 크기의 약 8~10%)
- 이중 외곽선 느낌 — 파란 외곽선 바깥에 얇은 밝은 테두리/글로우
- **카라오케 와이프 상태 확인됨**: 1행에서 "전에 자주"는 **흰색(#FFFFFF) 채움**, "비행했었잖아"는 연하늘 채움
  → 즉 **부른 부분 = 흰색, 안 부른 부분 = 연하늘**, 좌→우 진행형 와이프

**보조 가사 (로마자)**
- 산세리프 (Arial/Helvetica 계열), 볼드 아님 또는 세미볼드
- 채움색: 연노랑~크림 `#F2F0C4` 근처
- 외곽선: 어두운 올리브/브라운 `#4A3A10` 근처, 얇음 (2~3px 상당)
- 표기 방식: **음절 단위 띄어쓰기**, 어절 사이는 넓은 간격
  - 예: `jeo ne  ja ju  bi haeng hae seot jan a`
  - 예: `chin gu deul gwa  mal sseong jaeng i`

**본문 텍스트**
- 1행: `전에 자주 비행했었잖아` / `jeo ne  ja ju  bi haeng hae seot jan a`
- 2행: `친구들과 말썽쟁이` / `chin gu deul gwa  mal sseong jaeng i`

## preset-2-translation.png 시각 분석 (가로로 긴 크롭)

번역 자막 스타일. 배경은 애니메이션 영상 — **참고용, 오버레이 대상 아님**.

**레이아웃**
- 화면 **하단 중앙 정렬**, 2줄 스택
- 위: 원어(영어) / 아래: 번역(한글)
- 크기 비율 대략 **원어 0.6 : 번역 1.0** (번역이 더 큼 = 주 언어)

**원어 줄 (영어)**
- 볼드 **이탤릭** 산세리프 (Montserrat / Poppins ExtraBold Italic 류)
- 채움색: 노랑 `#F2C81E` ~ `#FFD400`
- 외곽선: 검정, 중간 두께 (약 4px 상당)
- 본문: `I just need someone in my life to give it structure`

**번역 줄 (한글)**
- 볼드 고딕, 이탤릭 아님 (Pretendard Black / Noto Sans KR Black 류)
- 채움색: 흰색 `#FFFFFF`
- 외곽선: 검정, 두꺼움 (약 6~7px 상당) + 약한 드롭섀도우
- 본문: `난 그냥 내 삶을 지탱해 줄 누군가가 있었으면 해`

**특징**
- 카라오케 와이프 **없음** — 줄 단위 통째로 표시/전환
- 두 줄 모두 가운데 정렬, 좌우 여백 대칭
